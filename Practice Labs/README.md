# Practice Labs

This directory contains small, self-created exercises and experiments that I built while learning Machine Learning.

Unlike the course assignments, these labs are not copied from any tutorial. They are my own implementations used to reinforce concepts, test ideas, and improve my problem-solving skills.

The goal of these labs is to practice concepts such as:

* Data preprocessing
* Feature engineering
* Model training
* Hyperparameter tuning
* Overfitting and underfitting
* Model evaluation
* Neural networks
* TensorFlow and scikit-learn workflows
* General experimentation

Some labs may be simple, incomplete, or intentionally repetitive. They represent my learning process and document how my understanding has evolved over time.

As I continue learning, new labs will be added and existing ones may be improved or revisited.
