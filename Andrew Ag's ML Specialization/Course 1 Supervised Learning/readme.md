# Supervised Machine Learning: Regression and Classification

> My notes, implementations, and projects from Course 1 of the **Machine Learning Specialization** by **Andrew Ng** and **DeepLearning.AI**.

## 📚 About This Course

This repository section documents my learning journey through the fundamentals of supervised machine learning.

Throughout this course, I learned how machine learning models can identify patterns in data and make predictions on unseen examples.

The course focuses on two major types of supervised learning:

- **Regression** — predicting continuous values
- **Classification** — predicting discrete categories

---

## 🎯 Learning Objectives

By completing this course, I learned how to:

- Understand the difference between supervised and unsupervised learning
- Build and train linear regression models
- Implement gradient descent from scratch
- Define and optimize cost functions
- Evaluate model performance
- Build binary classification models using logistic regression
- Prevent overfitting using regularization techniques
- Understand the bias-variance tradeoff
- Apply machine learning workflows to real-world problems

## 🧮 Key Concepts

| Concept | Description |
|----------|-------------|
| Feature | Input variable used by a model |
| Target Label | Output value the model predicts |
| Cost Function | Measures prediction error |
| Gradient Descent | Optimization algorithm for minimizing cost |
| Learning Rate | Controls update step size |
| Overfitting | Model memorizes training data |
| Regularization | Reduces model complexity |
| Classification | Predicting categories |
| Regression | Predicting continuous values |
