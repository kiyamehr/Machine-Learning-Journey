# NumPy Basics

> *Arrays, math, and the first step into numerical computing.*

This folder contains my early experiments with NumPy — the foundation of almost everything in ML. I'm learning how to work with arrays, perform operations, and understand distributions.

## ✅ Topics covered so far

- Creating and reshaping arrays
- Basic math operations (element-wise, dot product)
- Random distributions (exponential, normal)
- Indexing and slicing

## 🔜 Next topics

- Broadcasting
- Linear algebra basics (matrices, dot products)
- Saving/loading arrays

## 🧠 Why this matters

NumPy is how Python talks to numbers at scale. No NumPy = no real ML. So I'm starting here, slowly and intentionally.
