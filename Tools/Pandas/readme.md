Pandas is where data becomes useful. This folder tracks my progress learning DataFrames, Series, merging, cleaning, and everything that comes before a model sees the data.

- Creating DataFrames from dictionaries/lists
- Viewing data (head, tail, info, describe)
- Selecting columns and rows

## 🔜 Next topics

- Handling missing data (dropna, fillna)
- Filtering and conditionals
- Grouping and aggregation (groupby)
- Merging, joining, and concatenating

Real data is never clean. Pandas is my tool for making it usable before any model touches it.
